# TensorFlow

TensorFlow is an open-source machine learning and deep learning framework developed by Google. It is widely used for building, training, and deploying machine learning models for tasks such as image recognition, natural language processing, recommendation systems, and predictive analytics.

## Features

- Open-source and free to use
- Supports machine learning and deep learning
- GPU and TPU acceleration
- Cross-platform compatibility
- Easy integration with Python
- Model deployment for web, mobile, and cloud

## Technologies Used

- Python
- TensorFlow
- NumPy
- Pandas
- Matplotlib
- Jupyter Notebook

## Installation

Install TensorFlow using pip:

```bash
pip install tensorflow
```

Verify the installation:

```python
import tensorflow as tf
print(tf.__version__)
```

## Example

```python
import tensorflow as tf

# Create a constant tensor
hello = tf.constant("Hello, TensorFlow!")

print(hello.numpy())
```
